This is a web app that can compare playlists to show overlapping tracks,
requires local server like VScode live server or node.js


**Unfortunately user can only fetch playlists created personally due to Spotify API privacy rules, no blends no top songs no curated**


Usage: start a local server and run index.html. 
This will redirect you to a Oauth login page. you must login before being able to search playlists
